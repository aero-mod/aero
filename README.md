## Installing

Clone this repo to your machine:
```bash
$ git clone https://github.com/aero-mod/aero aero
```

Install deps and build it locally
```bash
$ pnpm i
$ pnpm build
```

Inject into discord
```bash
$ pnpm inject <channel>
```
